My New Post


  • hexoとは
    静的サイトジェネレータの1つ
    読み方分からん

  • シンタックス
    Qiitaのようにシンタックスハイライトされる

  • 手順

    1. ブログ生成
      $ hexo new “hoge”
      source/_posts 配下に .md ファイルが生成される

    2. ローカル環境で確認
      hexo server

    3. デプロイ
      $ hexo deploy -g
      これはgenerateの役目も担う
